# Smart Attendance Management System

A web-based attendance management system built using Python and Flask that allows faculty to mark attendance, students to view their records, and admins to monitor reports.

---

## Features

- Role-based login (Admin, Faculty, Student)
- Mark daily attendance
- View attendance reports
- Secure authentication
- Dashboard for quick access
- SQLite database for data storage
- Simple and clean UI

---

## Tech Stack

- Python
- Flask
- SQLite
- HTML, CSS
